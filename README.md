# FLO
- 안드로이드 FLO 클론 코딩입니다.
- 노래 재생을 service 로 구현하지 않아, 화면 별 전환 시 노래 끊김 현상이 있습니다. (추후에 업데이트할 예정)
- 사용기술 : roomDB, RecyclerView, ViewPager, Retrofit2, ViewModel
